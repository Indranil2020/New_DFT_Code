# Reference data (verification/references/)
- ACWF verification subset (all-electron equations of state; Bosoni et al., Nat. Rev. Phys. 2024 data)
- S22 (Jurecka et al.) and W4-11 subset (Karton et al.) with CCSD(T)-level reference values
- Surface DB: curated adsorption energies incl. CO/Pt(111) with its documented GGA failure
- Gauntlet-10: the 10-system smoke set every nightly runs (3 molecules, 2 slabs, 2 bulks incl. one
  metal, a-Si:H 512, 64-H2O box, one charged/spin case)
Every entry stores: source DOI, license, values, uncertainty, and the exact settings used to compare.
Adding a reference = PR with provenance; S9 approves.
